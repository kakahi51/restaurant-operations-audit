# Restaurant Operations Audit

## Background
This project analyzes operational data from a small-scale food business to identify discrepancies in inventory usage, sales records, and cost structure.

## Problem Statement
Manual tracking caused inconsistencies between raw material input and recorded sales, leading to inaccurate stock records and unclear operational performance.

## Data Overview
The dataset includes anonymized records of inventory inputs, daily sales, and cost components.

## Analysis Approach
- Inventory reconciliation
- Discrepancy detection
- Profitability analysis
